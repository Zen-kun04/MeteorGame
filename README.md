# Meteor 2D Game
> Made with Pygame

## Installation guide
```
git clone https://github.com/Zen-kun04/MeteorGame
cd MeteorGame
python -m venv venv
```
### If you are on Windows
```
venv\Scripts\activate.bat
```
### If you are on Mac/Linux
```
venv/Scripts/activate
```
### Finishing the installation
```
pip install -r requirements.txt
```

## Launch the game
```
python main.py
```

## Have fun ^^
